# 🚀 Smart Coders - Modern Programming Learning Website

Welcome to Smart Coders, a modern and responsive programming learning website built using HTML and Tailwind CSS.

---

## 🌐 Project Overview

This project is a fully responsive landing page designed for a programming learning platform.

### Features:
- 🔥 Premium Navbar with hover effects  
- 🎯 Hero Section with call-to-action  
- 💼 Services Section  
- 🏢 Clients Showcase  
- 💳 Pricing Plans (Basic, Pro, Enterprise)  
- 📩 Contact Form  
- 🧾 Professional Footer  

---

## 🛠️ Technologies Used

- HTML5  
- Tailwind CSS  
- JavaScript (Basic)
